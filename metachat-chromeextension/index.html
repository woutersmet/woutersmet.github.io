MetaChat

chat with fellow visitors of the site you are on

MetaChat is a Chrome plugin built with FireBase that shows a chat window where you can chat with people also on the same site.
To run the program:
  • Download and unpack the Metachat.zip file somewhere
  • go to chrome://extensions
  • Check 'developer mode'
  • Press 'load unpacked extension' and select the folder where you unpacked the metachat zip file

On this page you can see / participate in sites that people are on.

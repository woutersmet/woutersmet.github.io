Chat with fellow visitors of the site you are on

MetaChat is a Chrome extension that shows a chat window where you can chat with people also on the same site.
To run the program:
  • Download and unpack the Metachat.zip file somewhere
  • go to chrome://extensions to manage your extensions
  • Check 'developer mode' checkbox
  • Press 'load unpacked extension' button and select the folder where you unpacked the metachat zip file
  • Surf to a site, and click the icon in the top right to start chatting!
